# kdb dashboard library frontend

React + TypeScript starter for a kdb websocket backend.

## Quick start

```bash
cd frontend
cp .env.example .env
npm install
npm run dev
```

The app expects a websocket backend at `VITE_KDB_WS_URL`, defaulting to `ws://localhost:5000`.

For the longer integration notes and message contract, see `../docs/frontend/README.md`.
